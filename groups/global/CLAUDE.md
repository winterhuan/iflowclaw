# Global Memory

This file is shared across all groups and provides context that applies to all conversations.

## About iFlowClaw

iFlowClaw is a personal AI assistant that runs in containerized environments for security isolation. It uses the iFlow platform for AI capabilities.

## Capabilities

- Execute bash commands safely in containers
- Read and write files in mounted directories
- Schedule tasks for future execution
- Manage multiple chat groups independently

## Usage

Mention the assistant by name (default: @iFlow) to trigger a response in registered groups.
