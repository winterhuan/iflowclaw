# 内容团队 Skills 使用指南

> 本指南帮助你使用已安装的 6 个写作 Skills，构建高效的 AI 内容团队。

---

## 📋 Skills 概览

| Skill | 角色 | 核心功能 | 触发关键词 |
|-------|------|----------|------------|
| `content-strategy` | 内容策略师 | 内容规划、主题矩阵、内容日历 | 内容策略、内容规划、写什么 |
| `content-research-writer` | 深度内容作者 | 研究写作、引用添加、大纲迭代 | 写文章、深度文章、研究报告 |
| `copywriting` | 营销文案师 | 落地页、产品页、转化文案 | 营销文案、落地页、CTA |
| `social-content` | 社媒运营 | 社交媒体内容、平台策略 | LinkedIn帖子、Twitter、社媒 |
| `copy-editing` | 文案编辑 | 润色、语法检查、风格统一 | 编辑文案、润色、修改 |
| `content-humanizer` | 人性化专家 | 去AI味、品牌声音注入 | 太像AI、改自然点、人性化 |

---

## 🚀 快速开始

### 1. 内容策略规划

**使用场景**：规划季度/月度内容方向、主题矩阵

**示例提示词**：
```
帮我规划下个季度的内容策略，目标受众是科技创业者
```

```
帮我制定一个 B2B SaaS 产品的内容矩阵，包括博客、社媒、邮件
```

**输出内容**：
- 内容支柱（Content Pillars）
- 目标受众分析
- 内容类型规划
- 发布日历建议

---

### 2. 深度内容写作

**使用场景**：博客文章、白皮书、研究报告、教程

**示例提示词**：
```
帮我写一篇关于 AI 如何改变内容营销的深度文章，需要添加引用和数据支持
```

```
帮我创建一篇技术教程，主题是「如何使用 Claude Code 提高开发效率」
```

**工作流程**：
1. 分析主题 → 2. 创建大纲 → 3. 研究引用 → 4. 撰写初稿 → 5. 迭代优化

---

### 3. 营销文案创作

**使用场景**：落地页、产品页、定价页、广告文案

**示例提示词**：
```
帮我写一个 SaaS 产品的落地页文案，重点是提高免费试用转化率
```

```
帮我写一段产品介绍，突出「节省50%时间」的核心卖点
```

**核心原则**：
- 清晰 > 巧妙
- 利益 > 功能
- 一个页面一个目标

---

### 4. 社交媒体内容

**使用场景**：LinkedIn、Twitter/X、Instagram 等平台内容

**示例提示词**：
```
帮我写 5 条 LinkedIn 帖子，主题是远程工作的效率提升技巧
```

```
帮我创建一个 Twitter thread，介绍我们新产品的 5 个核心功能
```

**平台特点**：
| 平台 | 最佳用途 | 发布频率 | 推荐格式 |
|------|----------|----------|----------|
| LinkedIn | B2B、思想领导力 | 3-5次/周 | Carousel、故事 |
| Twitter/X | 实时、社区互动 | 3-10次/天 | Thread、热点评论 |
| Instagram | 视觉、品牌 | 1-2次/天 | Reels、图文 |

---

### 5. 文案润色编辑

**使用场景**：已有文案的优化、润色、质检

**示例提示词**：
```
帮我编辑这段营销文案，让它更有说服力：
[粘贴你的文案]
```

```
帮我检查这篇文章的语法和风格问题：
[粘贴你的文章]
```

**七步润色法**：
1. 清晰度检查 → 2. 简洁性 → 3. 有力度 → 4. 准确性
5. 一致性 → 6. 节奏感 → 7. 最终检查

---

### 6. 人性化改写

**使用场景**：AI 生成内容过于机械、需要注入品牌个性

**示例提示词**：
```
这段内容听起来太像 AI 写的了，帮我改得更自然：
[粘贴你的内容]
```

```
帮我给这篇文章注入我们品牌的个性——专业但不死板，有趣但不轻浮：
[粘贴你的文章]
```

**三步人性化**：
1. 检测 AI 特征 → 2. 移除机械模式 → 3. 注入品牌声音

## 💡 高级技巧

### 组合使用

**场景1：完整内容生产**
```

第二步：帮我写第一篇主题的深度文章
第三步：润色这篇文章
```

**场景2：营销活动**

```
第二步：基于落地页写一封邮件通知
```

创建 `marketing-context.md` 文件可以提供品牌背景，让所有 skills 更精准：

```

调性：专业、友好、简洁
受众：科技创业者、产品经理
产品：AI 效率工具
核心卖点：节省 50% 工作时间
```

## 🆘 常见问题

**Q: 如何选择用哪个 skill？**

A: 根据任务类型选择：
- 规划内容方向 → `content-strategy`
- 写长内容 → `content-research-writer`
- 写营销转化 → `copywriting`
- 写社媒 → `social-content`
- 优化已有内容 → `copy-editing`
- 去AI味 → `content-humanizer`

**Q: 可以同时使用多个 skills 吗？**

A: 可以按流程串联使用，每个 skill 完成一个环节。

**Q: 如何让输出更符合我的品牌？**

A: 在群组根目录创建 `product-marketing-context.md` 或 `marketing-context.md`，包含品牌调性、受众、产品信息。
